# BookStore

Functionality:
1. login
2. register
3. user can make orders and check current orders and history orders
4. admin panel for checking orders, making them completed and adding new books
5. search books by author,title, genre, pages

Used technologies:
1. Java EE
2. SQL
3. MySql
4. Bootstrap
5. HTML5
6. CSS
